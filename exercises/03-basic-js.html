The task
1. Following the example of the function `greetWorld`,
write a function `greetUser` that takes one argument (name) and alerts "Hello !".
2. Write function `askNameAndGreetUser` that prompts the user for their name
and then calls the function `greetUser` with the name as an argument.

All event listeners are already provided for you, you just need to write the functions.
Clicking on the "Greet the world" button will call the `greetWorld` function.
Clicking on the "Greet the user" button will call the `askNameAndGreetUser` function.
(both are already done for you)
